# 🏥 grupoF---TPF---Programacion3

Pre-entrega de Trabajo  Integrador de Programación 3 — Grupo F.  

Para esta pre-entrega se desarrolló el BREAD de pacientes, permitiendo registrar, consultar, actualizar y eliminar información dentro del sistema de gestión de turnos clínicos.

El proyecto está orientado a la administración de pacientes y turnos médicos mediante una API desarrollada con Node.js, Express y MySQL.
